福来宝德开发岗位招聘

WEB 开发工程师岗位职责：

  • 根据业务需求设计具体方案，包括系统架构，数据库设计，模块设计；
  • 负责公司信息系统和业务系统全线开发；
  • 测试，发布，并对已发布系统进行维护；
  • 开发系统内容自动化功能模板。

WEB 开发工程师职位要求：

  • 计算机相关基础知识扎实，熟悉常用数据结构与算法；
  • 熟悉Linux/UNIX操作系统和开发环境；
  • 熟悉 Ruby，PHP，Python 中一种以上开发语言，理解面向对象思想；
  • 熟练掌握HTML/CSS/JavaScript；
  • 熟练使用Git管理代码，熟悉 github 及开源软件；
  • 熟悉SQL语言和数据库基础知识；
  • 热爱编程，有Geek精神，乐于接受新鲜技术，技术包容性强；
  • 工作积极主动，具有强烈的责任感和团队合作精神；
  • 1年以上Ruby on Rails开发经验者优先。
